# Reliability & Survival Analysis — Machine Failure Data

## Overview
Statistical study of industrial machine failure patterns
using reliability theory, distribution fitting, survival
analysis and hypothesis testing.

## Key Highlights
- Fitted Weibull, Exponential & Log-Normal distributions
  to failure lifetime data
- Plotted empirical Survival/Reliability function R(t)
- Computed B10 and median machine life
- Kolmogorov-Smirnov goodness-of-fit test
- Hypothesis testing (T-test + Chi-square)

## Tools
Python | Pandas | NumPy | Matplotlib | Seaborn | Scipy

## Dataset
AI4I 2020 Predictive Maintenance — Kaggle (10,000 obs.)

## Key Results
- Best fit: [Weibull/Log-Normal] distribution
- Median machine life: [X] minutes tool wear
- Tool wear significantly higher in failed machines
  (p < 0.05)

## Relevance
Motivated by statistical inference research on lifetime
distributions — Kayal et al. (2019–2025)
